# Marissa Rounds Portfolio Website

## Folder Structure
- `index.html`
- `about.html`
- `resume.html`
- `portfolio.html`
- `writing.html`
- `leadership.html`
- `socials.html`
- `contact.html`
- `css/` for styles
- `js/` for JavaScript
- `images/` for headshot and future images
- `videos/` for the Who I Am video
- `docs/` for resume and writing sample PDFs

## To personalize the site fully
1. Replace `images/headshot-placeholder.svg` with the real headshot.  
   - Keep the filename the same, or change the image path inside `index.html`.

2. Add the final Who I Am video to the `videos` folder.  
   - Suggested filename: `who-i-am-placeholder.mp4`
   - If the filename changes, update the `<source>` path in `about.html`.

3. Add the resume and portfolio/writing PDFs to the `docs` folder.
   Use these filenames unless you want to edit the page links:
   - `marissa-rounds-resume.pdf`
   - `personal-brand-presentation.pdf`
   - `banned-works-paper.pdf`
   - `global-speech-research-paper.pdf`
   - `evidence-evaluation-paper.pdf`

4. Add the TED Talk link in `portfolio.html` where the placeholder link currently appears.

## Design Notes
This site uses a light blue, grey, and white palette and separates pages, styling, scripting, and media into their own folders to match your assignment directions.
